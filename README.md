# Uber Supply-Demand Gap Analysis

## Project Overview
This project analyzes the supply-demand gap in a ride-hailing service using Exploratory Data Analysis (EDA) on ride request data.

## Key Insights
- Airport has 2x more "No Cars Available" incidents than City
- City has 5.4x more cancellations than Airport
- Peak hours (evening/early morning) show highest demand-supply mismatch

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Plotly

## Recommendations
- Dynamic driver incentives at Airport during peak hours
- Investigate City cancellation reasons
- Predictive driver allocation algorithms
